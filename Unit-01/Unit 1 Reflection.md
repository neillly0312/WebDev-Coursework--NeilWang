# Unit 1 Reflection

[In Unit 1, I learned how the Internet and the web started. I found out that the U.S. Department of Defense created ARPANET, which was designed like a web with no central point so it could survive an attack. ARPANET was the first network to use packet switching, which means data is broken into small packets that can take different paths and then reassemble at the destination. I also learned about Elizabeth Feinler, who helped create the first domain name system, and Vint Cerf and Bob Kahn, who created TCP/IP, the foundation of today's Internet. In addition, I started learning HTML basics, including how to structure a web page using headings and paragraphs.

One challenge I faced was understanding how to type each tags and what each type represents. And I found that remembering tags is very difficult to me, so I need to check whther I use the correct tag or not.

This unit connects to web development because understanding how the Internet works helps me write better code. Knowing that browsers read HTML to display pages makes me more careful about writing clean and correct code from the beginning.]
